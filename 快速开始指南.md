# 🚀 数据可视化大屏维护功能 - 快速开始指南

## ⚡ 5分钟快速部署

### 第一步：执行数据库脚本（1分钟）

打开 MySQL 客户端，执行以下命令：

```sql
-- 1. 选择你的数据库
USE your_database_name;

-- 2. 执行脚本文件
SOURCE /path/to/sys_config_table.sql;

-- 或者直接复制粘贴 sys_config_table.sql 文件内容执行

-- 3. 验证是否成功
SELECT * FROM sys_config;
-- 应该看到 6 条配置记录
```

### 第二步：启动后端服务（1分钟）

```bash
cd backend
mvn spring-boot:run
```

等待看到：`Started VolunteerApplication in X seconds`

### 第三步：启动前端服务（1分钟）

```bash
cd frontend
npm run dev
```

看到：`Local: http://localhost:5173/`

### 第四步：登录测试（2分钟）

1. 打开浏览器访问：`http://localhost:5173`
2. 使用管理员账号登录
3. 点击左侧菜单"数据可视化大屏维护"
4. 修改配置并保存
5. 点击"预览大屏"查看效果

## 🔧 常见问题

### Q1: 数据库脚本执行失败？
**A:** 检查数据库连接是否正常，确保有创建表的权限。

### Q2: 后端启动报错？
**A:** 检查 `application.properties` 中的数据库配置是否正确。

### Q3: 前端无法访问后端接口？
**A:** 检查后端是否启动，端口是否为 8080，CORS 配置是否正确。

### Q4: 配置保存后大屏没有变化？
**A:** 刷新大屏页面（F5），配置会在页面重新加载时生效。

### Q5: 维护界面显示"加载配置失败"？
**A:** 检查数据库表是否创建成功，后端服务是否正常运行。
