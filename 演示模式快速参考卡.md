# 🎭 演示模式快速参考卡

## 🚀 一键启用

```javascript
// 复制到浏览器控制台执行
localStorage.setItem('DASHBOARD_MODE', 'demo'); location.reload()
```

## 🔄 一键关闭

```javascript
// 复制到浏览器控制台执行
localStorage.removeItem('DASHBOARD_MODE'); location.reload()
```

## 🗑️ 清除缓存

```javascript
// 复制到浏览器控制台执行
sessionStorage.removeItem('DASHBOARD_MOCK_DATA'); location.reload()
```

## 🐛 常见问题

**Q: 演示模式未生效？**  
A: 检查 localStorage，确保值为 'demo'

**Q: 数据不合理？**  
A: 清除 sessionStorage 缓存

**Q: 实时增长不工作？**  
A: 确认演示模式已启用，等待5秒
