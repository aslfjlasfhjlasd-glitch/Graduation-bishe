## 🔌 四、API 接口清单

### 管理员配置接口（6个）

1. ✅ `GET /api/admin/config/list`
   - 获取所有配置列表
   - 返回配置对象数组

2. ✅ `GET /api/admin/config/map`
   - 获取配置 Map
   - 返回 key-value 格式

3. ✅ `GET /api/admin/config/{configKey}`
   - 获取单个配置值
   - 根据键名查询

4. ✅ `PUT /api/admin/config/{configKey}`
   - 更新单个配置
   - 请求体包含新值

5. ✅ `POST /api/admin/config/batch-update`
   - 批量更新配置
   - 请求体为配置 Map

6. ✅ `POST /api/admin/config`
   - 添加新配置
   - 请求体为配置对象

### 大屏数据接口（2个）

1. ✅ `GET /api/dashboard/data`
   - 获取大屏综合数据
   - 包含统计数据和配置

2. ✅ `GET /api/dashboard/configs`
   - 获取大屏配置信息
   - 返回配置 Map
